# 目录结构

文件夹|功能
----------|-----------
db      | 数据库存储目录
models  | 数据库模型文件目录
node_modules|node第三方模块目录
public  | 公共文件目录（css、js、image）
routers | 路由目录
schemas | 数据库结构文件目录
views   | 模块视图文件目录
app.js  |入口文件

# 启动方式

```bash
netstat -apn | grep 4040  

netstat -tpln
# 查看端口

可以使用命令“node ~/example.js &”将项目置于后台运行。
```
